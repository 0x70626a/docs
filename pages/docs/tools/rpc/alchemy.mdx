# Getting your API keys from Alchemy

[![Discord](https://img.shields.io/discord/667044843901681675.svg?color=768AD4&label=discord&logo=https%3A%2F%2Fdiscordapp.com%2Fassets%2F8c9701b98ad4372b58f13fd9f65f966e.svg)](https://discord-gateway.optimism.io)
[![Twitter Follow](https://img.shields.io/twitter/follow/optimismFND.svg?label=optimismFND&style=social)](https://twitter.com/optimismFND)

This tutorial assumes you are completely new to Alchemy, but that you want to get applications and API keys to use with OP Mainnet or OP Goerli.


## Register

<Steps>
The first step is to register for a free Alchemy account.

### Step 1: Signup on Alchemy 
- Browse [to Alchemy](https://auth.alchemyapi.io/signup?redirectUrl=https%3A%2F%2Fdashboard.alchemyapi.io%2Fsignup%2F%3Freferrer_origin%3DDIRECT)
- Enter user information
- Click **Sign up**

   <img src="/assets/signup.png" />

### Step 2: Go to your e-mail and click **VERIFY EMAIL**

### Step 3: Fill out the requested information (team, type of project, etc.)

   <img src="/assets/team-form.png" width="300"/>

### Step 4: When asked for chain, select **Optimism** and click **Next** 

   <img src="/assets/select-op.png" width="300"/>

### Step 5: Select the free plan and click **Next**

### Step 6: Either enter payment details or click **Skip for now**

   <img src="/assets/payment-details.png" width="300"/>

### Step 7: Enter where you heard about Alchemy and click **Let's Build!**

   <img src="/assets/lets-build.png" width="300"/>
</Steps>

## Create new applications

The next step is to create applications and get their API keys.

### Create a new OP Goerli application

<Steps>
### Step 1: Browse to the [Alchemy dashboard](https://dashboard.alchemyapi.io/)

### Step 2: Click **+CREATE APP**

### Step 3: Specify a name, select the chain **Optimism** and the network **Optimism Goerli**

   <img src="/assets/create-app-l2-testnet.png" width="300"/>

### Step 4: Click **CREATE APP**

### Step 5: Click **VIEW KEY** in the Testnet-L2 line

   <img src="/assets/viewkey.png" width="500"/>

### Step 6: Copy the API Key

   <img src="/assets/copykey.png" width="300"/>

   Alternatively, if you need a URL, copy the HTTPS definition.
</Steps>

### Create a new Goerli application

Follow the same instructions as for OP Goerli, but select the chain **Ethereum** and the network **Goerli**.

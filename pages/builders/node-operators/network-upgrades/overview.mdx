# Network Upgrade Overview

This section has information on how to upgrade your Mainnet and Testnet nodes for new network upgrades.

## Upgrade Process

<Steps>
  ### Devnet

  *   Devnet Upgrade Notice Period
  *   Upgrade Activates on Devnet
  *   Baking on Devnet

  ### Testnet

  *   Testnet Upgrade Notice Period
  *   Upgrade Activates on Testnet
  *   Baking on Testnet

  ### Mainnet

  *   Governance Voting Review Period
  *   Governance Voting Period
  *   Cut Mainnet Release
  *   Mainnet Upgrade Notice Period
  *   Upgrade Activated
</Steps>

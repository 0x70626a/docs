# Delta Network Upgrade

The following information is how you can prepare your nodes for the Delta upgrade.

## Mainnet Information

<Callout type="warning">
    The Delta mainnet network upgrade is not scheduled yet.
</Callout>

## Testnet Information 

<Callout type="warning">
    *   The Delta testnet network upgrade for OP Goerli will activate at **1703116800 Thu Dec 21 00:00:00 UTC 2023**.
    *   The Delta testnet network upgrade for OP Sepolia will activate at **1703203200 Fri Dec 22 00:00:00 UTC 2023**.
</Callout>

### Required Action

You must upgrade your nodes to the following versions prior to 

*   Upgrade your `op-node` to version: [v1.4.0](https://github.com/ethereum-optimism/optimism/releases/tag/op-node%2Fv1.4.0)
*   Upgrade your `op-geth` to version: [v1.101304.2](https://github.com/ethereum-optimism/op-geth/releases/tag/v1.101304.2)

### Verifying Your Configuration on OP Goerli & OP Sepolia

You can verify your node is configured correctly with the following:

*   `op-node` will log its configuration at startup

#### OP Goerli

*   Check that Delta time is set to `1703116800` in the op-node logs

#### OP Sepolia

*   Check that Delta time is set to `1703203200` in the op-node logs
